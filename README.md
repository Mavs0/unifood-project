# unifood-frontend

## 🧑‍💻 Como Rodar o Projeto Localmente

**Clone o repositório**

```bash
[git clone git@github.com:UniFood-Team/unifood-frontend.git](https://github.com/Mavs0/unifood-project)
cd unifood-project
```
**Instale as dependências:**
``` bash
npm install
```
**Inicie o servidor de desenvolvimento:**
``` bash
npm run dev
```
